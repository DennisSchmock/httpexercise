EXERCISE 2: FORM BASED SECURITY IN THIS LINK:

This demo uses formbased security.
  • Formbased security stores the credentials in a session on the server, tied to a unique session-id.
  • The information about the session is sent between client and server upon each request.
  • When logged off the server "destroys" the session, removing the credentials.
